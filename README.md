# xamarin-playground

Some testes made to learn Xamarin Form

Everything from Jan 2016


## Dependencies:

* Xamarin Studio 5.10
* Android SDK
* Xcode (Mac)


## Quick Start

* Clone the repository
```sh
$ git clone https://github.com/Moggi/xamarin-playground.git
```

* Open in Xamarin Studio

* You're good to go
