# GCA - Gerenciador de Centro Academico


## Dependencies:

* Xamarin Studio 5.10
* Android SDK
* Xcode (Mac)

## Quick Start

* Clone the repository
```sh
# Don't forget to change "<yourUserName>"
$ git clone https://<yourUserName>@bitbucket.org/adpt/gca-mobile.git
```

* Open in Xamarin Studio

* You're good to go
